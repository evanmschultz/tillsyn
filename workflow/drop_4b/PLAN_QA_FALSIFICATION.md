## 2. CONFIRMED Counterexamples — Summary + Fix Direction

1. **WC-A1: Lease-revoke cascade missing.** Plan §2.4.8 default assumption ("session revoke is sufficient") contradicts code — capability leases live in `s.repo.RevokeCapabilityLeasesByScope`, not the autent backend. **Fix:** 4b.5 must explicitly revoke both session AND lease via `RevokeAllCapabilityLeases`. Add test `TestRevokeSessionForActionItemRevokesBothSessionAndLease`.
2. **WC-A2: Multi-session leak.** Plan §2.4.4 "picks the first non-revoked session." Retries / fix-builder cycles leave additional sessions live. **Fix:** iterate over ALL matching sessions and revoke each via `errors.Join` aggregation. Update test scenario 7.
3. **CW-A2: Drop 4a R1 + R2 publish gap not covered.** Plan misses the publishActionItemChanged additions for `RestoreActionItem`/`RenameActionItem`/`ArchiveActionItem`/`ReparentActionItem`/`ImportSnapshot`. Without them, 4b.7's subscriber never wakes on those write paths — auto-promotion is partial. **Fix:** add explicit Wave C droplet(s) — either 4b.8 (publish gap fix) or fold into 4b.7's scope. Each write-path success branch calls `s.publishActionItemChanged(projectID)`.

## 3. PASS-WITH-NIT Findings

1. **WA-A2: side-effect rollback policy** — add doc-comment to 4b.2 noting policy is per-gate; 4b's gates need none, Drop 4c gates need rollback design.
2. **WA-A3: null-byte / binary-output handling** — `tailOutput` should `strings.ToValidUTF8` before storing in `BlockedReason`.
3. **WA-A5: empty-Packages silent-success** — flip default to fail-empty (zero-cost change).
4. **WC-A5: path-traversal audit** — verify Drop 4a 4a.5 normalization actually rejects `../` patterns; surface to plan-QA-proof.
5. **WC-A9: subscriber amplifies 4a.19 stub Hylla leakage** — worklog NIT for 4b.7 builder; if Drop 5 reveals harm, fast-track Drop 4c F.7.10.
6. **WC-A10: `hylla_reingest` gate semantics** — drop the gate from 4b.7 entirely; defer to Drop 4c F.7. Closed-enum `IsValidGateKind` keeps `hylla_reingest` as a known name; gate runner returns `ErrGateNotRegistered` if a template binds it pre-Drop-4c. Fails loud rather than silently passes on stub.
