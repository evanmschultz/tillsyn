## 2. Refuted Attacks

- **2.1 — `GateRule` leak (per L6).** REFUTED. `schema.go` contains the literal token `GateRules` only inside the doc comment on `Template` ("The struct intentionally has no GateRules field"). LSP `documentSymbol` enumerates exactly four struct types — `Template`, `KindRule`, `ChildRule`, `AgentBinding` — and one constant (`SchemaVersionV1`). No `GateRule` symbol. No `GateRules` field on `Template`. L6 honored.

- **2.2 — `AgentBinding` field count (per § 19.3 lines 1653-1656).** REFUTED. LSP enumerates 11 fields with the exact names + types the spec lists: `AgentName string` (123), `Model string` (127), `Effort string` (132), `Tools []string` (137), `MaxTries int` (141), `MaxBudgetUSD float64` (145), `MaxTurns int` (148), `AutoPush bool` (152), `CommitAgent string` (156), `BlockedRetries int` (160), `BlockedRetryCooldown time.Duration` (164). Every field present, every type matches.

- **2.3 — TOML tag drift.** REFUTED. Read of every field declaration confirms each has a `toml:"snake_case_name"` struct tag matching the field's snake-case spelling: `schema_version`, `kinds`, `child_rules`, `agent_bindings`, `owner`, `allowed_parent_kinds`, `allowed_child_kinds`, `structural_type`, `when_parent_kind`, `create_child_kind`, `title`, `blocked_by_parent`, `when_parent_structural_type`, `agent_name`, `model`, `effort`, `tools`, `max_tries`, `max_budget_usd`, `max_turns`, `auto_push`, `commit_agent`, `blocked_retries`, `blocked_retry_cooldown`. Twenty-four unique snake-case tags; no camelCase, no missing tag, no duplicates within the same struct.

- **2.4 — Round-trip fixture coverage.** REFUTED. `TestTemplateTOMLRoundTrip` populates every nested map + slice with a real concrete entry. `Kinds[domain.KindBuild]` carries non-empty `AllowedParentKinds` + `AllowedChildKinds` slices and a non-zero `StructuralType`. `ChildRules[0]` populates all five fields including `WhenParentStructuralType`. `AgentBindings[domain.KindBuild]` populates all 11 `AgentBinding` fields with non-zero values: `AgentName` "go-builder-agent", `Model` "opus", `Effort` "high", `Tools` `[]string{"Edit","Write","Bash"}`, `MaxTries` 3, `MaxBudgetUSD` 2.5, `MaxTurns` 50, `AutoPush` true, `CommitAgent` "commit-agent", `BlockedRetries` 2, `BlockedRetryCooldown` `30 * time.Second`. The fixture is sufficient — a zero-only fixture would not be.

- **2.5 — Map key marshaling for typed-string aliases.** REFUTED EMPIRICALLY. `Kinds map[domain.Kind]KindRule` and `AgentBindings map[domain.Kind]AgentBinding` use `domain.Kind` (a `type Kind string` alias). `pelletier/go-toml/v2 v2.2.4` (per `go.mod:66`) round-trips typed-string-alias keys by treating them as plain strings during inline-table key emission and decoding back into the typed alias via reflection-based key conversion. The round-trip test passed `reflect.DeepEqual(original, decoded)` end-to-end with one populated entry under `domain.KindBuild` ("build") in each map. If go-toml/v2 had silently dropped the typed-key entry on decode, the decoded map would have been nil-or-empty and `DeepEqual` would have failed. It did not — verdict: typed-string-alias map keys round-trip cleanly. No CONFIRMED counterexample.

- **2.6 — `time.Duration` round-trip.** REFUTED. The fixture populates `BlockedRetryCooldown: 30 * time.Second` (a non-zero `time.Duration`, which is `int64` nanoseconds). go-toml/v2 marshals `time.Duration` via its native serializer (string form `"30s"` or numeric form depending on encoder config). Round-trip passed `reflect.DeepEqual` — the decoded `time.Duration` exactly equals 30 seconds. No drift.

- **2.7 — `StructuralType` typed-alias round-trip.** REFUTED. `KindRule.StructuralType` is set to `domain.StructuralTypeDroplet` ("droplet") and `ChildRule.WhenParentStructuralType` is set to the same. Round-trip equality holds. Same typed-string-alias mechanism as `domain.Kind`; same outcome.

- **2.8 — Empty-vs-nil drift.** REFUTED. Every map and slice in the fixture is pre-populated with at least one element (nested slices `AllowedParentKinds`, `AllowedChildKinds`, `Tools` each have ≥ 1 element; both top-level maps `Kinds` and `AgentBindings` have one entry; `ChildRules` slice has one entry). The test comment at line 42-44 explicitly documents the rationale ("populated with at least one element so the assertion does not trip on the nil-vs-empty-slice asymmetry"). No nil-vs-empty trap.

- **2.9 — YAGNI.** REFUTED. `schema.go` declares: 1 constant + 4 structs + 24 struct fields + zero functions, methods, constructors, validators, `String()` accessors, `MarshalText()`/`UnmarshalText()` shims, `Validate()` methods, or any other behavior. The package imports only `time` (for `Duration`) and `internal/domain` (for `Kind` + `StructuralType`). No `init()`. The build is the smallest concrete shape that satisfies the spec.

- **2.10 — Doc comment completeness.** REFUTED. Read of schema.go confirms a doc comment on (a) the package itself (lines 1-14, including canonical-source pointers), (b) the constant `SchemaVersionV1` (lines 22-26), (c) every struct (`Template` 28-37, `KindRule` 58-60, `ChildRule` 84-87, `AgentBinding` 112-119), and (d) every struct field (every field in all four structs). No undocumented top-level declaration. Comments are full sentences and reference canonical sources (PLAN.md § 19.3, methodology §11, L6/L7/N4 fix tags).

- **2.11 — `SchemaVersion` as first field on `Template`.** REFUTED. `Template.SchemaVersion string \`toml:"schema_version"\`` is declared at line 42, the first field of the struct. A v2 parser sniffing the `schema_version` key at the root will see it.

- **2.12 — Struct-tag conflict.** REFUTED. Per-struct tag inventories: `Template` (4 tags: `schema_version`, `kinds`, `child_rules`, `agent_bindings` — distinct); `KindRule` (4 tags: `owner`, `allowed_parent_kinds`, `allowed_child_kinds`, `structural_type` — distinct); `ChildRule` (5 tags: `when_parent_kind`, `create_child_kind`, `title`, `blocked_by_parent`, `when_parent_structural_type` — distinct); `AgentBinding` (11 tags: `agent_name`, `model`, `effort`, `tools`, `max_tries`, `max_budget_usd`, `max_turns`, `auto_push`, `commit_agent`, `blocked_retries`, `blocked_retry_cooldown` — distinct). No collisions inside a struct.

- **2.13 — Out-of-package leak.** REFUTED. `schema.go`'s only imports are `time` (stdlib) and `github.com/evanmschultz/tillsyn/internal/domain`. `schema_test.go` adds `reflect`, `testing`, `time`, `github.com/pelletier/go-toml/v2` (test-only), and `internal/domain`.
