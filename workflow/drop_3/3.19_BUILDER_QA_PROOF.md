## 3. Findings

No blocking findings. Two informational observations recorded for falsification-sibling and orchestrator awareness; neither blocks PASS.

- **3.1 (informational, accepted) — Test 7 (Supersede) is structurally deferred, not silently dropped.** The droplet description's PLAN.md acceptance text enumerates "Supersede(Owner=STEWARD, AuthRequestPrincipalType=agent) → REJECTED (per 5.C.13)" as PLAN case 7, but no Supersede API exists in the action-item domain or adapter today. The builder correctly deferred with rationale rather than fabricating an API to satisfy a test. Routing: when the future droplet that lands the Supersede API arrives, that droplet's acceptance set must include the same `assertOwnerStateGate` invocation; the gate helper is already in place and only needs a wrapper to consume it. PLAN.md `[origin: 5.C.13]` already flags this as a forward dependency.

- **3.2 (informational) — `assertOwnerStateGate` returns `ErrAuthorizationDenied` even on no-caller-in-context paths over STEWARD-owned items.** The helper at `app_service_adapter_mcp.go:911-926` rejects when `app.AuthenticatedCallerFromContext(ctx)` returns `(_, false)` and the item is STEWARD-owned. The doc-comment frames this as "treat as non-steward". This is conservative (fail-closed) and matches the gate's intent — anyone who reaches the gate without a real authenticated caller cannot prove they are steward, so the gate must reject. Tests `TestAssertOwnerStateGateMoveActionItemAgentRejected` and `TestAssertOwnerStateGateMoveActionItemStateAgentRejected` exercise this path indirectly because `withMutationGuardContext` rebuilds the caller from the actor tuple's `AuthRequestPrincipalType` field, so a `stewardGatedActor("agent")` does land a non-empty caller — but the unguarded fallback inside `assertOwnerStateGate` is exercised by no test. Falsification sibling may want to look at this surface (no production-code attack vector visible to the proof reviewer because every adapter entry point goes through `withMutationGuardContext`, but worth a sanity pass).
