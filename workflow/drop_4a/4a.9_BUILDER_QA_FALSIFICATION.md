### 2.5 Test-count delta divergence — Unknown

Builder claims `mage ci` post-4a.9 reports 1949 tests vs 1929 at 4a.8 baseline = +20 delta. Reviewer's structural inspection of the diff finds:

- 1 new test function in domain (`TestNewActionItemEndCommitTrim`, 6 subtests)
- 1 new test function in MCP (`TestActionItemMCPEndCommitRoundTrip`, 5 subtests)
- 1 new test function in snapshot (`TestSnapshotActionItemEndCommitRoundTrip`, 4 subtests)
- 1 new test function in snapshot (`TestSnapshotActionItemEndCommitLegacyFormatCompatibility`, no subtests)
